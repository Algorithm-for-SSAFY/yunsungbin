### 문제 설명

<p>자연수 n이 주어졌을 때, GCD(n, k) = 1을 만족하는 자연수 1 ≤ k ≤ n 의 개수를 구하는 프로그램을 작성하시오.</p>

### 입력 

 <p>첫째 줄에 자연수 n (1 ≤ n ≤ 10<sup>12</sup>)이 주어진다.</p>

### 출력 

 <p>GCD(n, k) = 1을 만족하는 자연수 1 ≤ k ≤ n 의 개수를 출력한다.</p>

### 접근 방법
교재에서 처음으로 본 문제라 "오일러 공식"을 이용해다 한다는 것은 알고 있었습니다.

하지만, 오일러 공식을 모를 때 이 문제를 풀 수 있을까?에 대해서 생각해보았고, 오일러 공식을 안쓰고 접근해보았습니다.

일단, 자연수 N에 대해서 10^12의 최댓값을 가질 수 있으므로 O(N) 연산은 무조건 시간 초과가 날 것으로 생각했습니다.

해서, 최대 O(루트N)의 복잡도를 가져야 한다는 것을 알 수 있었습니다.

1. GCD = 1을 구하란 것은 자연수 N과 서로소을 구하라는 것 이다.
2. 자연수 N을 소인수 분해 한다. 이후, 소수 들을 모아놓는다.
3. 2번에서 수집한 소수들을 이용해서 뺄 것 이다.
4. 1 ~ N 중의 숫자 중, 예를 들어 소수 2를 뺴고자 하면, 2의 배수는 모두 서로소로서 탈락이기 때문에 N개의 후보 중, 2의 배수만큼을 모두 제외한다.
5. 4번의 과정을 2번에서 수집한 소수들로 모두 수행하면 정답을 도출할 수 있따.

### 회고
- 오일러 공식을 모르게 접근 했어도, 소인수 분해와 서로소의 개념만 좀 알면 응용해서 풀 수 있었던 것 같다.
